# Employee Leave Management System

## Project Description
The **Employee Leave Management System** is a web-based application designed to manage employee leave requests efficiently. This system helps organizations automate the leave request and approval process, reducing manual work and improving transparency between employees and managers.

The application allows employees to submit leave requests and managers to review, approve, or reject them. It also maintains leave balances and displays employee leave schedules through a calendar view.

---

## Technologies Used

- **Backend:** Java, Spring Boot
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** MySQL
- **Build Tool:** Maven
- **IDE:** IntelliJ IDEA / Eclipse / VS Code

---

## Features

### 1. User Authentication
- Secure login system
- Role-based access (Employee and Manager)
- Password encryption for security

### 2. Employee Features
- Submit leave request
- View leave request status
- Check leave balance
- View leave calendar

### 3. Manager Features
- View employee leave requests
- Approve or reject leave requests
- Add comments on leave requests

### 4. Leave Management
- Track vacation and sick leave
- Automatic leave balance update
- Maintain leave history

### 5. Leave Calendar
- Calendar view of employee leave
- Highlights approved leaves
- Shows employee availability

---

## System Workflow

1. User registers in the system.
2. User logs in with credentials.
3. Employee submits a leave request.
4. Manager reviews the request.
5. Manager approves or rejects the request.
6. Leave balance is updated automatically.
7. Approved leave is shown in the leave calendar.
